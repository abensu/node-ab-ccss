# node-ab-ccss


## 简介

本程序用于生成 css-sprite （图片精灵）的图片合成和生成对应 css 文件

提倡**傻瓜式**的用法，使用详情请查看 [demo 用例](https://github.com/abensu/node-ab-ccss/tree/master/demo)

**适合团队或个人开发的任何阶段**


## 依赖

[images](https://github.com/zhangyuanwei/node-images)


## 安装

```
npm install node-ab-ccss
```
